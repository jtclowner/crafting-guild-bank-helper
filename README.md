# Crafting Guild Bank Helper

A small RuneLite Plugin Hub plugin that highlights the Crafting Guild bank chest clickbox after using a Crafting cape or Max cape Crafting Guild teleport.

![Crafting Guild Bank Helper example](/images/demo.gif)

## Features

- Shows a temporary predicted bank chest clickbox while the teleport animation is playing.
- Switches to the real bank chest clickbox once the player lands at the Crafting Guild.
- Clears the highlight after clicking the bank chest or moving away from the landing tile.
- Configurable highlight colour, fill, opacity, and outline width.

## Notes

This plugin only draws an overlay. It does not click, automate input, or interact with the game on the player's behalf.
